Projects

  • Online Library Python

    A simple media library web application where users can browse a list of items and view details such as title, author, date, and category in a clean, single‑page interface.

    A software program for organizing a digital collection of media materials is called the Online Library project. Through a desktop graphical user interface that connects to a backend HTTP API, users engage with the system. Books, movies, and magazines are examples of media goods; each has characteristics like name, author, publication date, and genre. This project is driven by the desire to demonstrate proficiency in a number of advanced programming subjects, including file-based persistence, GUI programming with Tkinter, REST API development with Flask, modular software design, and automated testing with Pytest. The project incorporates several Python libraries and adheres to best practices for documentation, error handling, and code organization. The full software development lifecycle is covered in this study, including requirements analysis, architectural design, implementation across several modules, testing techniques, and operational considerations. All functional requirements have been satisfied, and the solution has been successfully implemented, tested, and documented using automated test cases.
